# DashNest 🧠

A calm and clean WPF personal dashboard built using C# and .NET 8.0.

### Features
- 📝 Notes
- ✅ To-Do List
- 📅 Event Reminders
- 💾 Offline, local JSON storage
- 🎨 Minimal glass-inspired UI

### How to Run
1. Clone repo
2. Open in Visual Studio
3. Build & run!
